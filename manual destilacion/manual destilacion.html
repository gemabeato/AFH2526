Ejecutado en Mac mini (Apple Silicon / MPS)

Paso 1: Configuración del Entorno

El primer desafío fue preparar el Mac mini para Deep Learning utilizando la aceleración por hardware MPS (Metal Performance Shaders) en lugar de CUDA (NVIDIA).

Comandos ejecutados:

# Creación del entorno aislado
python3 -m venv venv
source venv/bin/activate

# Instalación de librerías optimizadas
pip install torch torchvision transformers datasets accelerate

Paso 2: El Script de Destilación

Creamos el archivo destilar.py. El objetivo era usar DistilBERT (Profesor) para enseñar a BERT-Tiny (Estudiante) usando la divergencia KL.

Error Crítico 1: Argumentos inesperados

TypeError: compute_loss() got an unexpected keyword argument 'num_items_in_batch'

Solución Aplicada

Actualizamos la firma de la función añadiendo **kwargs para hacerla compatible con las nuevas versiones de Hugging Face.

Error Crítico 2: Incompatibilidad de Modelos

TypeError: forward() got an unexpected keyword argument 'token_type_ids'

Solución Aplicada

El Profesor no soportaba ciertos inputs del Estudiante. Filtramos el diccionario de entrada antes de pasarlo al modelo gigante.

Paso 3: Entrenamiento en Mac mini

Una vez depurado el código, lanzamos el entrenamiento. El uso de device="mps" fue clave para lograr velocidad.

  • Velocidad: ~5.83 iteraciones/segundo.
  • Tiempo Total: ~18 minutos.
  • Reducción de Pérdida: De 1.47 a 0.86.
Prueba 1: El entrenamiento corriendo fluidamente en Apple Silicon.

Paso 4: Resultados Finales

Al finalizar las 3 épocas, logramos una compresión masiva con una pérdida mínima de calidad.

Paso 5: Validación y Barrera de Idioma

Al principio, el modelo falló al probar frases en español (clasificaba todo como negativo). Descubrimos que el Profesor solo sabía inglés.

Al cambiar las pruebas al inglés, el modelo demostró su verdadero potencial:

Prueba 3: Clasificación perfecta de Sentimiento (Positivo/Negativo) con alta confianza.

Paso 6: El Duelo Final (Comparativa)

Enfrentamos al Profesor (66M) contra el Estudiante (4M) con acertijos lógicos y sarcasmo.

Conclusiones del Experimento:

  • Sarcasmo: El estudiante aprendió a detectar ironía ("Great... flat tire").
  • Sesgos: El estudiante heredó los errores de lógica del profesor (Doble negación).
  • Límite Cognitivo: El estudiante falló en frases largas con sentimientos mixtos debido a su menor capacidad de atención.
